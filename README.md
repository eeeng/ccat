# 마지막 인사: 다시, 너를 만나러

A p5.js-based browser game with story and stage sections.

## Repository structure

- `index.html` - main game entry point
- `src/` - game source files (`game.js`, `story.js`, `sketch.js`)
- `styles/` - stylesheet (`style.css`)
- `libs/` - p5 libraries used by the game
- `assets/images/` - image assets used by the game
- `tools/` - helper scripts and utilities

## Run locally

Open `index.html` in a web browser.

> For best results, run via a local web server if your browser blocks local file access for image assets.

## Notes

- `src/story.js` and `src/game.js` now use `assets/images/` for image loading.
- The project is organized for clean GitHub upload and easy maintenance.
